# Book Store Database — SQL & Relational Modeling Project

Учебно-практический проект по проектированию и реализации реляционной базы данных
для книжного магазина с использованием SQL.

Проект охватывает полный цикл работы с БД:
от логического проектирования схемы до написания аналитических SQL-запросов
различной сложности.

---

## Описание предметной области

База данных описывает процесс продажи книг и включает следующие сущности:

- **Покупатели** (ФИО, район проживания, персональная скидка)
- **Магазины** (название, район размещения, комиссионные)
- **Книги** (название, стоимость, склад, количество)
- **Покупки** (дата, покупатель, магазин, книга, количество, сумма)

---

## Цели проекта

- Спроектировать логическую модель БД по заданному предметному описанию
- Реализовать таблицы с ограничениями целостности данных
- Заполнить БД тестовыми данными
- Разработать аналитические SQL-запросы разного уровня сложности
- Освоить подзапросы, агрегации и операции над множествами

---

## Реализация

### Проектирование БД
- Создание таблиц с использованием:
  - `PRIMARY KEY`
  - `FOREIGN KEY`
  - `NOT NULL`
  - `UNIQUE`
  - `CHECK`
- Обоснованный выбор типов данных для каждого поля

### Данные
- Заполнение таблиц тестовыми значениями (`INSERT`)
- Проверка корректности данных (`SELECT *`)
- Коррекция данных (`UPDATE`, `DELETE`)

---

## SQL-запросы

Проект включает запросы **двух уровней сложности**.

### Уровень 1
- Простые и составные `SELECT`
- Фильтрация и сортировка данных
- Соединения таблиц (`JOIN`)
- Работа с датами
- Модификация данных:
  - пересчёт суммы покупки с учётом скидки
  - добавление и расчёт комиссионных магазинов

### Уровень 2 (аналитические запросы)
- Подзапросы
- `IN / NOT IN`
- `ALL / ANY`
- `EXISTS / NOT EXISTS`
- `UNION`
- Агрегатные функции (`COUNT`, `SUM`, `AVG`)
- Группировка (`GROUP BY`, `HAVING`)

Примеры решаемых задач:
- анализ покупательской активности
- сравнение сумм покупок между клиентами
- поиск покупателей по сложным условиям
- аналитика по магазинам и районам
- отчёты по обороту и количеству продаж
